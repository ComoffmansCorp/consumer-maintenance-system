Electrician Work SaaS Demo

Демонстрационная панель для защиты проекта. Через нее можно показать multi-tenant модель, авторизацию, tenant-администрирование, задачи исполнителя и простую бизнес-аналитику.

SUPER_ADMIN Логин: superadmin
Пароль: admin123
TENANT_ADMIN Tenant: demo-energy
Логин: tenantadmin
Пароль: tenant123
ELECTRICIAN Tenant: demo-energy
Логин: electrician1
Пароль: electric123

1. Логин

Для platform user tenantCode оставь пустым.

2. Токен и профиль

После входа токен и профиль текущего пользователя будут доступны для всех остальных действий.

Пока нет активной сессии

3. Platform Admin

Доступно только для SUPER_ADMIN.

4. Tenant Admin

Создание пользователей внутри компании.

5. Исполнитель

Просмотр задач и изменение статуса.

6. Акт осмотра

Минимальный JSON для демонстрации создания акта.
